# Stonks

Algorithmic Trading Platform integrating Forester prediction models with Golden Ratio Strategy.

## Full Vision
Stonks is designed as a three-tier architecture algorithmic trading platform:
1. **Model Layer**: Core machine learning and trading strategy implementation (Completed).
2. **API Layer**: FastAPI backend to serve model predictions and signals (Coming soon).
3. **Frontend Layer**: React + Plotly user interface for interactive dashboards (Coming soon).

## Current Status
- **Model**: Complete. The Forester model and Golden Ratio backtesting pipeline are fully operational.
- **API**: To be built.
- **Frontend**: To be built.

## Tech Stack
- **Model**: Python, Scikit-Learn, Pandas, yfinance
- **API**: FastAPI (Planned)
- **Frontend**: React, Plotly (Planned)
